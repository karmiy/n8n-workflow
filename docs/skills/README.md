## Pin 数据

有时想固定某个接口得数据时，可以用 Pin 将数据固定，这样后续节点都可以使用这个测试数据（如大模型花钱请求的数据，不想每次 test 都去发一次消耗 token，就可以 Pin 固定这个节点）
